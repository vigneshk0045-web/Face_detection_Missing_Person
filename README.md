# Face Recognition Surveillance System & Communication Platform

## Overview
The Face Recognition Surveillance System is an AI-based application designed to identify missing persons and criminals using facial recognition technology. The system processes images and live camera feeds to detect matches and notify users in real time.

## Features
- Face recognition using computer vision techniques  
- Identification of missing persons and criminals  
- User and admin interfaces for image upload and verification  
- Real-time alert notifications on successful match detection  
- Secure data storage and management  

## Tech Stack
- **Programming Language:** Python  
- **Libraries & Frameworks:** OpenCV  
- **Database:** MongoDB  

## System Highlights
- Supports image-based and live camera feed recognition  
- Tested under different lighting conditions to improve accuracy  
- Designed for reliability with proper data handling and alert mechanisms  

## Future Enhancements
- Improve model accuracy using deep learning techniques  
- Add real-time CCTV stream integration  
- Enhance user interface for better usability  
- Implement role-based access control
